# Task 2: Temperature Monitoring System 🌡️

## Overview

This project is a temperature monitoring system developed using Arduino. The system reads temperature values from a temperature sensor and provides real-time monitoring with status indication based on temperature conditions.

The project demonstrates basic IoT concepts such as sensor data acquisition, processing, and output indication.

## Objectives

* Monitor temperature values in real-time
* Interface temperature sensor with Arduino
* Display temperature status based on threshold values
* Understand basic embedded system development

## Components Used

* Arduino Board
* Temperature Sensor
* LED Indicators
* Connecting Wires
* Breadboard
* Power Supply

## Software Used

* Arduino IDE
* Embedded C / Arduino Programming

## Working Principle

1. The temperature sensor collects the surrounding temperature data.
2. Arduino processes the sensor input.
3. The system compares the temperature value with predefined limits.
4. Based on the temperature level, the system indicates the status.

## Temperature Conditions

* SAFE Temperature: Normal operating temperature range
* WARNING Temperature: High temperature detected condition
