# 📂 Completed Tasks

## ✅ Task 2: IoT-Based Temperature Monitoring and Overheat Alert System

### Description

A sensor-based temperature monitoring system developed using Arduino Uno, TMP36 temperature sensor, LCD display, LED, and buzzer.

The system measures temperature continuously and provides alerts when the temperature exceeds a predefined threshold value.

### Technologies Used

* Arduino Uno
* TMP36 Temperature Sensor
* 16×2 LCD Display
* Tinkercad Simulation
* Embedded C/C++ Programming

### Features

* Real-time temperature monitoring
* LCD temperature display
* LED warning indicator
* Buzzer alert system
* Threshold-based temperature detection
